## Getting Started

### Prerequisites

- Any modern web browser
- Git (for GitHub deployment)

Simply open `index.html` in your web browser

#### Option 2: Using live server
```bash

```

#### Option 3: Using npm (if Node.js installed)
```bash

npm run dev
```

## GitHub Pages Deployment

This website is optimized for GitHub Pages! üöÄ

### Quick Deploy Steps:

1. **Create Repository**: Create a new public repository named `personal-website` on GitHub
2. **Push Code**: 
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/personal-website.git
   git push -u origin main
   ```
3. **Enable Pages**: Go to Settings > Pages > Source: Deploy from a branch > main
4. **Visit Site**: Your site will be live at `https://YOUR_USERNAME.github.io/personal-website`

üìñ **Detailed Guide**: See `GITHUB_DEPLOYMENT.md` for complete instructions

## Customization

### Personal Information

Update the following in `index.html`:
- Your name in the hero section
- About section content
- Skills and proficiency levels
- Projects and their details
- Contact information

### Styling

The website uses TailwindCSS. You can customize:
- Colors: Modify the gradient colors in the `<style>` section
- Fonts: Change the Google Font import
- Layout: Adjust Tailwind classes throughout the HTML

### Adding Projects

To add new projects:
1. Duplicate an existing project card in the projects section
2. Update the image, title, description, and tech stack
3. Add actual links to live demo and GitHub repository
